# Проект 7-го спринта

🚀 Проект 7-го спринта
📋 Описание
Проект представляет собой ETL-пайплайн для обработки геоданных и расчета витрин. Данные проходят через четыре слоя обработки, результатом являются готовые витрины для аналитиков.

🏗 Архитектура приложения
📦 Слои данных
Слой	Расшифровка	Назначение	Описание
RAW 📥	Raw Data	Сырые данные	Исходные события в формате поставщика
ODS 🔄	Operational Data Store	Предобработка	Данные с партиционированием по дате
DDS ⚙️	Detail Data Store	Обогащение	Промежуточные данные с координатами и городами
DML 📊	Data Mart Layer	Витрины	Готовые данные для аналитиков
📊 Витрины данных (слой DML)
Витрина	Пиктограмма	Назначение	Содержание
UsersMart	👥	Информация о пользователях	ID пользователя, город, координаты
ZonesMart	🏙️	Статистика по городам	Сообщения, реакции, подписки за неделю/месяц
FriendsMart	🤝	Рекомендации друзей	Общие каналы, переписка, географическая близость
💾 Формат хранения
Параметр	Значение
Путь	/user/username/project7/dml/
Формат	Parquet (колоночное хранение)
Обновление	Ежедневно в 00:00
✅ Преимущества Parquet
💪 Эффективное сжатие — экономия места в HDFS

📐 Сохранение схемы — типы колонок не теряются

⚡ Высокая производительность — оптимизирован для аналитических запросов

📅 Партиционирование — удобная фильтрация по дате

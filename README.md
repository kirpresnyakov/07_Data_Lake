# Проект 7-го спринта

🚀 Проект 7-го спринта
📋 Описание
Проект представляет собой ETL-пайплайн для обработки геоданных и расчета витрин. Данные проходят через четыре слоя обработки, результатом являются готовые витрины для аналитиков.

🏗 Архитектура приложения
📦 Слои данных
Слой	Назначение	Описание
RAW 📥	Сырые данные	Исходные события в формате поставщика
ODS 🔄	Operational Data Store	Предобработанные данные с партиционированием по дате
DDS ⚙️	Detail Data Store	Промежуточные данные (обогащенные координатами и городами)
DML 📊	Data Mart Layer	Готовые витрины для аналитиков
📊 Витрины данных (слой DML)
Витрина	Назначение	Содержание
UsersMart 👥	Информация о пользователях	ID пользователя, город, координаты
ZonesMart 🏙️	Статистика по городам	Сообщения, реакции, подписки за неделю/месяц
FriendsMart 🤝	Рекомендации друзей	Общие каналы, переписка, географическая близость
